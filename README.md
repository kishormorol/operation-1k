# 🚀 Operation 1K — 1,000 Hours to AI Engineer

A personal study tracker built to mass 1,000 hours of focused learning in 6 months. Covers LeetCode, ML fundamentals, LLM engineering, RAG, agents, system design, and interview prep.

**Live:** [operation-1k.vercel.app](https://operation-1k.vercel.app)

## What's Inside

**26 weeks. 656 tasks. 253 LeetCode problems. 143 resource links. Every day dated.**

| Month | Weeks | Focus |
|-------|-------|-------|
| 1 | W1–W4 | Arrays, Hashing, Two Pointers, Sliding Window, Binary Search, Stack, Linked Lists, Trees + Python, Git, FastAPI, first LLM API calls |
| 2 | W5–W8 | Trees DFS/BFS, Tries, Heaps, Backtracking, Graphs + Embeddings, Chunking, Vector DBs, full RAG pipeline |
| 3 | W9–W13 | Advanced Graphs, 1D/2D DP, Intervals, Greedy + Agents, LangGraph, Evals, Workflows |
| 4 | W14–W17 | Hard LC mix, Design problems + Docker, Auth, Monitoring, Caching, System Design |
| 5 | W18–W21 | ML theory deep dive + 2 deployed portfolio projects |
| 6 | W22–W26 | Company research, mock interviews, interview sprint |

## Features

- **Plan** — 26 scrollable weeks, each broken into 7 days with checkable tasks. Every LeetCode problem links directly to the problem page. Every resource links to the doc/tutorial/video.
- **Time Tracking** — When you complete a task, log how many minutes it took. See time per task, per day, per week, and per category.
- **Routine** — Auto-detects today and shows the daily schedule (gym, study blocks, class, driving, walk).
- **Stats** — Progress by week, by category (DSA, ML, LLM, BUILD, DESIGN, MOCK), total hours, time tracked.
- **Dates** — Every day shows the exact calendar date. Today is highlighted. Week tabs show date ranges.
- **Persistent** — All data saved in localStorage. Survives refreshes.
- **Mobile-first** — Works on phone and desktop. Add to home screen for app-like experience.

## Deploy Your Own

```bash
git clone https://github.com/kishormorol/operation-1k.git
cd operation-1k
npm install
npm run dev
```

Open `localhost:5173`. To deploy on Vercel:

1. Push to GitHub
2. Go to [vercel.com](https://vercel.com) → Import repo → Deploy

## Add to Phone

**iPhone:** Open URL in Safari → Share → Add to Home Screen  
**Android:** Open in Chrome → Menu → Add to Home Screen
